Todos os meus projetos

Fit-Club

Desenvolvi uma página para uma academia, usando HTML e CSS para a estrutura e estilo. A tornai responsiva e funcional com JavaScript, incluindo validações de formulários e um carrossel.

Car-Spotting

Galeria de imagens utilizando JSON, HTML, Sass e JavaScript. Quando o usuário clica em uma das fotos na página inicial, é redirecionado para outra com uma geração aleatória de imagens do JSON.

Cafeterias-Bh

Criei uma landing page estática usando Bootstrap para um projeto da faculdade. A página tem a função de apresentar cafeterias na região de Belo Horizonte e seus principais produtos.

Glamour Joias

E-commerce utilizando Bootstrap, destacando-se com uma seção de produtos e uma dedicada à compra. Este site estático foi criado com o intuito de aprimorar minhas habilidades na utilização das classes do Bootstrap.

Lista de terefas

Meu primeiro projeto com React uma lista de tarefas interativa. Oferece a opção de adicionar tarefas, especificar categorias (estudos,trabalho,pessoal), marcar como concluídas e excluir individualmente ou todas as tarefas.

Clima

Aplicação simples em HTML, CSS e JavaScript, onde os usuários podem verificar o clima, a umidade e a previsão do tempo de qualquer cidade desejada. A aplicação é responsiva para garantir uma experiência consistente em diferentes dispositivos.

Login

Aplicação para aprimorar conhecimentos em JavaScript, apresentando um sistema simples de login. Os usuários podem cadastrar informações e realizar login usando tanto o e-mail quanto o número de telefone, sendo os dados armazenados localmente com o auxílio do LocalStorage.
